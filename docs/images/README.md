# Screenshots for the repository README

Add PNG (or WebP) screenshots here so the root [`README.md`](../../README.md) can embed them.

## Recommended exports

| Asset | Suggested size | Notes |
|-------|----------------|--------|
| Main window | **1440–1600 px wide**, PNG @2x-friendly | Light mode; crop window chrome only |
| Optional dark | Same | If the UI supports dark mode well |
| USB / progress | Same width family | Shows the “happy path” |

Keep **transparent** backgrounds only if the README uses a matching GitHub theme; otherwise **opaque** window backgrounds read more reliably.

## Naming

Examples (uncomment the matching `<img>` in `README.md`):

- `screenshot-home.png` — primary hero image  
- `screenshot-usb.png` — write / progress screen  

After files exist, uncomment the `<img>` block in the root `README.md` and set `alt` text for accessibility.

## Social preview (GitHub Open Graph)

You can use `social-preview.png` as the repository social preview image (configured in **GitHub repo settings**, not via git).
