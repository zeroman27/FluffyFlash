# Skills для проекта Wist

## Локальные (репозиторий)

| Папка | Назначение |
|-------|------------|
| `wist-xcode` | Пути к `Wist.xcodeproj`, сборка, скрипты, подпись |
| `obsidian-wist-vault` | Obsidian vault, Roadmap / Architecture / сессии |
| `ui-ux-pro-max` | UI/UX справочник (в т.ч. SwiftUI в данных) |

## Установлено с GitHub (MIT)

Источник: [Swift Agent Skills](https://github.com/twostraws/swift-agent-skills) (Paul Hudson / сообщество).

| Папка | Репозиторий | Описание |
|-------|-------------|----------|
| `swiftui-pro` | [twostraws/swiftui-agent-skill](https://github.com/twostraws/swiftui-agent-skill) | Обзор и ревью SwiftUI: API, доступность, производительность |
| `swift-concurrency-pro` | [twostraws/Swift-Concurrency-Agent-Skill](https://github.com/twostraws/Swift-Concurrency-Agent-Skill) | async/await, actors, Sendable, изоляция |
| `swift-testing-pro` | [twostraws/Swift-Testing-Agent-Skill](https://github.com/twostraws/Swift-Testing-Agent-Skill) | Тесты на Swift Testing |

Обновление: снова клонировать нужный репозиторий и заменить соответствующую папку, либо `git pull` в клоне и скопировать каталог `*-pro`.

**Заметка:** в текстах скиллов иногда упоминается iOS как эталон — проект **macOS**; правила всё равно применимы к SwiftUI и Swift на Mac.
