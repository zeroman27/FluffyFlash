# Видение Wist

> Кратко: **для кого** продукт, **какую проблему** решает, **чем** отличается.

## Для кого

- Пользователям macOS, которым нужен **официальный образ Windows 10/11** и **загрузочная USB** без отдельной машины с Windows.

## Проблема

- Скачать ISO и записать установщик на флешку с Mac неочевидно: лимит FAT32 на файлы >4 ГБ (`install.wim`), нужны `diskutil`, `wimlib`, понимание UEFI.
- Готовые инструменты разрознены или не open-source в удобном виде.

## Почему мы

- **Нативное** приложение SwiftUI, без песочницы там, где нужны системные утилиты.
- **Пошаговый сценарий** с одной главной цепочкой (загрузка UUP → сборка ISO → запись на USB) и понятной навигацией без длинных инструкций (см. [[Architecture]]).
- **Интерфейс на нескольких языках** (в т.ч. иврит и арабский): строки в каталоге, исходный текст в коде — английский.
- **Открытый исходник**, бесплатно для всех.
- Опора на проверенные идеи: UUP ([CrystalFetch](https://github.com/TuringSoftware/CrystalFetch)), разбиение WIM ([WinDiskWriter](https://github.com/TechUnRestricted/WinDiskWriter) — идея split + FileManager-подобное копирование; у нас свой Swift-код, без копирования их GPL-исходников).

## Не цели (на старте)

- Подписанный дистрибутив вне GitHub / не App Store pipeline.
- Поддержка всех кастомных репаков ISO — фокус на официальных путях (UUP / Microsoft).
